# SAT - Sentiment Analysis Project

## Project Overview
This project performs sentiment analysis on text data using machine learning techniques.

##  Technologies Used
- Python
- Google Colab
- Scikit-learn
- Pandas
- NumPy

## Dataset
The dataset contains text reviews labeled as positive or negative.

##  Model Used
- Logistic Regression / Naive Bayes / etc.
- Accuracy: 87% (example)

##  How to Run
1. Open the notebook in Google Colab.
2. Run all cells sequentially.
3. The model will train and display results.

##  Files Included
- SAT.ipynb
- dataset.csv
